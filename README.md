## Instalação 

É necessário instalar dependências de ambos frontend e backend. Em suas respectivas pastas, rode:

```npm install```

## Execução 

Para executar o projeto, rode no backend

```npx json-server --watch db.json```

E no frontend

```npx expo start```

Para testes, use a versão web (aperte W no terminal)

Certifique que o cliente consegue acessar o backend (seja por meio de um túnel, ou só estando dentro da mesma rede)
